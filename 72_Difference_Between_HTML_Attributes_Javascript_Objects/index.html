HTML Attributes vs. DOM Properties

The test below monitors the input field for changes, showing the differences between the following values:

  • HTMLInputElement.value
  • HTMLInputElement.defaultValue
  • HTMLInputElement.getAttribute("value")
  • HTMLInputElement.getAttribute("defaultValue")

It is important to know the difference as they do NOT return the same values. People use the term "attribute" for both when they flat out are not the same thing.

HTMLInputElement.getAttribute
Note you COULD set a "defaultvalue" attribute in the markup, however that would not validate as there is no such attribute! In that case if the value="" and made up defaultvalue="" were different, .getAttribute("defaultvalue") would disagree with .defaultValue! The latter always being equal to the value attribute in the markup.
